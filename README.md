# TWM
## Summer project
Talk With Me - online chat

### Technologies
* sockets
* threading
* sqlite db
* pyqt5, pyqt5 editor
* git
* AES encryption
* select

### Programming languages used
* Python
* SQL

### Env. Variables
* client: AES_PUBLIC_KEY - 16 digits key
* server: 
  * AES_PUBLIC_KEY - same as the client's key
  * MAIL_ADDRESS - for sending emails to clients
  * MAIL_PASSWORD
